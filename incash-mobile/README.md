# Incash Mobile

The mobile application for the Incash ecosystem, built with React Native and Expo.

## Getting Started

### Prerequisites

- Node.js
- npm or yarn
- Expo Go app on your mobile device (or an emulator)

### Installation

1. Navigate to the mobile directory:
   ```bash
   cd incash-mobile
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Running the App

Start the development server:

```bash
npm start
```

- **Run on Android**: `npm run android`
- **Run on iOS**: `npm run ios`
- **Run on Web**: `npm run web`
